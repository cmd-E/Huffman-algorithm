-w - Input to encode
-f - File with input to encode
-w = -f, but from file. If -w and -f flags are defined, -f will be executed

===Encoding and decoding with predefined occurrences/probabilities===
-p - Path to file where custom occurrences for all symbols in word are defined
-e - Input to encode
-d - Input to decode
-prob - Available if -p is defined. Occurrences for symbols are treated as probabilities
-s - Available if -p is defined. Changes default separator (-) on user's one
